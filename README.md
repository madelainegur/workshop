# Royal Pet Gallery

A vintage-inspired mini shop website for selling framed pictures of dogs and cats dressed as kings and queens in luxurious gold frames.

## Features

- **Real Pet Portraits**: Beautiful dog and cat images from Unsplash
- **Majestic Hero Image**: Stunning banner with dogs and cats together
- **Sparkle Animations**: Enchanting gold sparkles falling in the background
- **Shopping Cart**: Full cart functionality with item management
- **Checkout System**: Complete checkout process with form validation
- **Vintage Gold Design**: Lavish gold color palette throughout
- **Responsive Layout**: Works perfectly on all devices
- **Smooth Animations**: Hover effects and transitions
- **Cart Icon**: Visual cart indicator with item count

## Features Breakdown

### Shopping Cart
- Click the cart icon in the header to view your cart
- Add multiple items of the same product
- Remove items from the cart
- View total price in real-time

### Checkout Process
- Fill out shipping and payment information
- View order summary before placing order
- Form validation ensures all fields are completed
- Success confirmation after order placement

### Design Elements
- **All Gold Theme**: Complete gold color scheme throughout
- Sepia-toned pet images for vintage aesthetic
- Ornate gold-style frames
- Elegant typography using Google Fonts (Cinzel & Lato)
- Modal overlays for cart and checkout
- Smooth transitions and hover effects

## Running the Website

Simply open `index.html` in your web browser. No build process or server required.

## Files

- `index.html` - Main HTML structure with product listings
- `styles.css` - All styling including cart and checkout modals
- `script.js` - Shopping cart and checkout functionality
- `README.md` - This file

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts
