# Royal Canine Gallery

A vintage-inspired mini shop website for selling framed pictures of dogs dressed as kings and queens.

## Features

- Classic vintage design with modern functionality
- Responsive grid layout for product showcase
- Smooth hover effects and animations
- Elegant typography using Google Fonts (Cinzel & Lato)
- Add to cart functionality with notifications

## Design

The website uses a vintage aesthetic with:
- Sepia and burgundy color palette
- Ornate gold-style frames
- Classic serif fonts for headings
- Clean, simple layout

## Running the Website

Simply open `index.html` in your web browser. No build process or server required.

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and design
- `script.js` - Interactive functionality
- `README.md` - This file
